블루무브 닥스에 로그인하기

블루무버

🔐 로그인에서 '블루무버 계정으로 로그인' 버튼을 클릭하세요.

  • 블루무버 계정(예: example@bluemove.or.kr)으로 로그인하세요.
  • 블루무브의 Google Drive 공유 드라이브에 대한 권한이 있어야 합니다.
  • 블루무브의 Slack 워크스페이스에 가입된 상태여야 합니다.
  • 블루무버 얼럼나이의 경우 게스트로 로그인해주시기 바랍니다.
게스트

🔐 로그인에서 'Google 계정으로 로그인' 버튼을 클릭하세요.

  • 개인 Google 계정(예: example@gmail.com)으로 로그인하세요.
  • 회사나 학교에서 제공하는 Google Workspace 계정(예: example@university.ac.kr)으로는 로그인할 수 없습니다.
  • Google 계정이 없을 경우 Google 계정 만들기에서 새로 만들 수 있습니다.

회원 정보 등록하기

블루무브 닥스 첫 로그인 시 🔐 로그인에서 성과 이름, 휴대전화 번호를 입력하고 '등록하기' 버튼을 클릭하세요.

  • 성과 이름은 실명으로, 휴대전화 번호는 실제 사용 중인 번호를 입력하세요.
  • 회원 정보는 등록 이후에도 수정할 수 있습니다.
  • 회원 정보가 정상적으로 등록되면 블루무버에게는 Slack 메시지가, 게스트에게는 이메일이 발송됩니다.
  • 첫 로그인 시 로그인을 취소하면 블루무브 닥스에 임시 저장된 Google 계정 정보가 삭제됩니다.

회원 정보 수정하기

💙 내 계정에서 '수정하기' 버튼을 클릭하세요.

  • 이메일 주소는 수정할 수 없습니다.
  • 성과 이름은 실명으로, 휴대전화 번호는 실제 사용 중인 번호를 입력하세요.
  • 회원 정보는 유효 문서(예: 미제출 문서, 제출된 문서, 반려된 문서)가 존재하지 않는 한 언제든 수정할 수 있습니다.
    (2021.05. 업데이트 예정)

회원 정보 삭제하기

💙 내 계정에서 '삭제하기' 버튼을 클릭하세요.

  • 블루무버의 경우 생성했던 문서함이 모두 삭제된 상태여야 합니다.
  • 게스트의 경우 유효 문서(예: 미제출 문서, 제출된 문서, 반려된 문서)가 존재하지 않는 상태여야 합니다.
  • 회원 정보 삭제 시 블루무브 닥스에 저장된 Google 계정 정보와 해당 계정을 통해 생성된 공지사항, 댓글이 함께 삭제됩니다.
  • 승인 및 반환된 문서는 회원 정보 삭제의 영향을 받지 않습니다.
  • 회원 정보가 정상적으로 삭제되면 블루무버에게는 Slack 메시지가, 게스트에게는 이메일이 발송됩니다.
  • 삭제된 회원 정보는 복원할 수 없습니다.

Google 계정 만들기

Google 계정 만들기에서 Google 계정을 새로 만들 수 있습니다.

  • Google 계정이 없을 경우 블루무브 닥스를 이용할 수 없습니다.
  • 블루무버 활동회원의 경우 테크 매니저에게 문의해주시기 바랍니다.

Google 계정 프로필 사진 변경하기

Google 계정에서 Google 계정 프로필 사진을 변경할 수 있습니다.

공지사항 업데이트 시 알림 받기

📢 공지사항 또는 해당 공지사항 페이지에서 (북마크) 버튼을 클릭하세요.

  • 북마크에 있는 공지사항이 수정되면 블루무버에게는 Slack 메시지가, 게스트에게는 이메일이 발송됩니다.
  • 북마크에 있는 공지사항의 제목이 수정될 경우 수정 전 제목과 수정 후 제목을 모두 확인할 수 있습니다.
  • 공지사항에 댓글이 작성될 경우 작성자는 북마크 추가 여부와 관계없이 Slack 메시지 알림을 받을 수 있습니다.
  • 북마크는 💙 내 계정에서 확인할 수 있습니다.

공지사항 댓글에서 멘션하기

해당 공지사항 페이지 댓글란에서 멘션할 사람의 댓글을 찾아 '멘션하기'를 클릭하세요.

  • 공지사항 댓글에서 멘션된 블루무버에게는 Slack 메시지가, 게스트에게는 이메일이 발송됩니다.
  • 멘션 대상은 해당 공지사항에 댓글을 작성한 사람으로 한정됩니다.

문서 반려하기

해당 문서함 페이지 '미결재 문서'에서 반려할 문서를 찾아 '반려하기'를 클릭하세요.

  • ⏳(모래시계) 아이콘이 표시되면 브라우저를 닫지 말고 작업이 완료될 때까지 기다려주시고 60초 이내에 작업이 완료되지 않을 경우 새로고침 후 다시 시도해주시기 바랍니다.
  • 생성자(검토자)가 입력한 반려 사유는 해당 문서 작성자(제출자)에게 전달됩니다.
  • 문서가 정상적으로 반려되면 생성자(검토자)와 블루무버 작성자(제출자)에게는 Slack 메시지가, 게스트 작성자(제출자)에게는 이메일이 발송됩니다.
  • '미결재 문서'는 생성자(검토자)에게만 표시됩니다.

문서 승인 및 반환하기

해당 문서함 페이지 '미결재 문서'에서 승인(또는 반환)할 문서를 찾아 '승인(또는 반환)하기'를 클릭하세요.

  • ⏳(모래시계) 아이콘이 표시되면 브라우저를 닫지 말고 작업이 완료될 때까지 기다려주시고 60초 이내에 작업이 완료되지 않을 경우 새로고침 후 다시 시도해주시기 바랍니다.
  • 블루무버 작성자(제출자)의 문서 승인 시 해당 문서가 최종 디렉토리로 이동되며 문서에 대한 권한은 해당 공유 드라이브 또는 최종 디렉토리의 설정에 따르게 됩니다.
  • 게스트 작성자(제출자)의 문서 반환 시 생성자(검토자)는 해당 문서에 대한 모든 권한을 잃게 되고 작성자(제출자)는 수정 및 삭제 권한을 얻게 됩니다.
  • 문서가 정상적으로 승인(또는 반환)되면 생성자(검토자)와 블루무버 작성자(제출자)에게는 Slack 메시지가, 게스트 작성자(제출자)에게는 이메일이 발송됩니다.
  • '미결재 문서'는 생성자(검토자)에게만 표시됩니다.
